# Troubleshooting (problemas comuns)

## VS Code não compila após copiar o exemplo
Isso pode acontecer por um bug/estado do VS Code após copiar/substituir pastas do projeto.

✅ Faça:
- Abra `main.c`
- Copie tudo, apague e cole novamente
- Compile e rode

## Evite abrir a pasta do exemplo antes de criar o projeto
Para reduzir bugs, crie o projeto primeiro em uma pasta vazia e só depois copie os arquivos do exemplo para dentro do projeto.

## Se o atalho não funcionar
- Tente abrir uma nova janela (**Ctrl + Shift + N**)
- Use **Ctrl + Shift + P** e procure os comandos “Genesis Code: …”
