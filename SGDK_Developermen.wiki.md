# Como publicar esta Wiki no GitHub

Este pacote contém páginas prontas para uma Wiki do GitHub.

## Opção A — Repositório Wiki (recomendado)
1. No GitHub, abra o repositório principal.
2. Clique em **Wiki** → **Create the first page** (se ainda não existir).
3. No seu computador, clone o repositório da Wiki (ele termina com `.wiki.git`).
4. Copie estes arquivos `.md` para dentro do repositório clonado.
5. Commit e push.

## Opção B — Docs dentro do repositório
Se você preferir, pode colocar estas páginas em uma pasta `/docs/wiki/` dentro do repositório principal.
